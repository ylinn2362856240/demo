
##环境准备
####1.安装node.js
1. *[nodejs下载地址](http://nodejs.cn/download/)
2. 安装nodejs完成完后，运行 node -v （如果显示 v.x.x.x  <x表示你安装的nodejs版本号>，恭喜你，nodejs成功安装，接下来就可以下一步，运行命令行！）

* 全局安装gulp
~~~
npm install gulp -g
~~~

* 全局安装bower  妇
~~~
npm install bower -g
~~~

####2.安装git
1. [安装git教程](https://git-scm.com/book/zh/v1/%E8%B5%B7%E6%AD%A5-%E5%AE%89%E8%A3%85-Git)
